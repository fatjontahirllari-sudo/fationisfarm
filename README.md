# 🐑 FationiFarm

App i thjeshtë për menaxhimin e fermës:
- regjistron qumështin
- ruan historikun
- llogarit fitimin

## 📱 Si përdoret
1. Hap app
2. Shkruaj litrat e qumështit
3. Shtyp "Ruaj"
4. Shiko historikun

## 💻 Si ta hapësh
1. Shkarko file `fationifarm.html`
2. Hap me Chrome
3. Add to Home Screen

## 🔧 Teknologjitë
- HTML
- CSS
- JavaScript
